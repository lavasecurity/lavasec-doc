# 支持与帮助

需要帮助、发现 bug，或想反馈问题时，可在这里找到相应途径。

## App 里有什么不正常？ {#something-not-working-in-the-app}

请使用 **App 内的错误报告**——这是最快捷的方式，它会自动附上我们所需的诊断信息，并通过设计默认将数据降至最少（绝不包含你浏览的域名）。报告将由团队分类处理。

## 发现了安全漏洞？ {#found-a-security-vulnerability}

**请勿公开提交 issue。** 请发送邮件至 **security@lavasecurity.app**，并附上问题描述、影响范围和复现步骤。有关范围和披露流程，请参阅 iOS 客户端的[安全策略](https://github.com/lavasecurity/lavasec-ios/blob/main/SECURITY.md)。_（邮箱地址待确认。）_

## 对开源客户端有疑问？ {#question-about-the-open-source-client}

Lava 的 iOS 客户端是开源的。如有技术问题、构建问题或功能建议，欢迎在 [**lavasecurity/lavasec-ios**](https://github.com/lavasecurity/lavasec-ios/issues) 提交 issue。

## 这些文档里有错？ {#something-wrong-in-these-docs}

每个页面底部都有一个 **「这个页面有帮助吗？」** 的提示 —— 选 *「可以更好」* 会打开一个预先填好的 issue，提交到 [**lavasecurity/lavasec-doc**](https://github.com/lavasecurity/lavasec-doc/issues)。欢迎纠错，也欢迎提 pull request。
