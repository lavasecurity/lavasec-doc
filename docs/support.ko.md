# 지원 및 도움말

도움이 필요하거나, 버그를 발견했거나, 무언가를 신고하고 싶을 때 찾아갈 곳입니다.

## 앱에서 무언가 작동하지 않나요?

**앱 내 버그 신고**를 사용하세요 — 가장 빠른 방법이며, 설계상 데이터를
최소화한 상태로 필요한 진단 정보를 함께 보냅니다(방문한 도메인은
절대 포함되지 않습니다). 신고는 팀에서 분류하여 처리합니다.

## 보안 취약점을 발견하셨나요?

**공개 이슈를 열지 마세요.** **security@lavasecurity.app** 으로 설명, 영향,
재현 단계와 함께 이메일을 보내주세요. 범위와 공개에 관해서는 iOS 클라이언트의
[보안 정책](https://github.com/lavasecurity/lavasec-ios/blob/main/SECURITY.md)
을 참조하세요. _(주소는 확정 예정입니다.)_

## 오픈 소스 클라이언트에 대한 질문이 있으신가요?

Lava의 iOS 클라이언트는 오픈 소스입니다. 기술적인 질문, 빌드 문제,
기능 아이디어가 있다면
[**lavasecurity/lavasec-ios**](https://github.com/lavasecurity/lavasec-ios/issues) 에서
이슈를 열어주세요.

## 이 문서에 잘못된 부분이 있나요?

모든 페이지 하단에는 **"이 페이지가 도움이 되었나요?"** 안내가 있습니다.
*"개선이 필요해요"* 를 선택하면
[**lavasecurity/lavasec-doc**](https://github.com/lavasecurity/lavasec-doc/issues) 에
미리 작성된 이슈가 열립니다.
수정 및 풀 리퀘스트를 환영합니다.
