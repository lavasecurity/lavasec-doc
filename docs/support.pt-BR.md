# Suporte e ajuda

Para onde ir quando você precisa de ajuda, encontrou um bug ou quer relatar algo.

## Algo não está funcionando no app?

Use o **relatório de bug no app** — é o caminho mais rápido e anexa os
diagnósticos de que precisamos, com dados minimizados por design (os domínios
que você navega nunca são incluídos). Os relatórios são triados pela equipe.

## Encontrou uma vulnerabilidade de segurança?

**Não abra uma issue pública.** Envie um e-mail para **security@lavasecurity.app** com uma
descrição, o impacto e os passos para reproduzir. Consulte a
[política de segurança](https://github.com/lavasecurity/lavasec-ios/blob/main/SECURITY.md)
do cliente iOS para escopo e divulgação. _(Endereço a confirmar.)_

## Dúvida sobre o cliente open-source?

O cliente iOS da Lava é open source. Para dúvidas técnicas, problemas de build ou
ideias de funcionalidades, abra uma issue em
[**lavasecurity/lavasec-ios**](https://github.com/lavasecurity/lavasec-ios/issues).

## Algo errado nesta documentação?

Toda página tem um aviso **"Esta página foi útil?"** no rodapé — escolher
*"Poderia ser melhor"* abre uma issue pré-preenchida em
[**lavasecurity/lavasec-doc**](https://github.com/lavasecurity/lavasec-doc/issues).
Correções e pull requests são bem-vindos.
