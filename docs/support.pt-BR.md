# Suporte e ajuda {#support--help}

Para onde ir quando você precisa de ajuda, encontrou um problema ou quer relatar algo.

## Algo não está funcionando no app? {#something-not-working-in-the-app}

Use o **relatório de problema dentro do app** — é o caminho mais rápido e ele anexa
os dados de diagnóstico de que precisamos, com a coleta de informações reduzida ao
mínimo por princípio (os domínios que você acessa nunca são incluídos). Os relatos
são analisados pela equipe.

## Encontrou uma falha de segurança? {#found-a-security-vulnerability}

**Não abra um relato público.** Envie um e-mail para **security@lavasecurity.app** com uma
descrição, o impacto e os passos para reproduzir. Consulte a
[política de segurança](https://github.com/lavasecurity/lavasec-ios/blob/main/SECURITY.md)
do cliente iOS para saber o escopo e como funciona a divulgação. _(Endereço a confirmar.)_

## Tem uma dúvida sobre o cliente de código aberto? {#question-about-the-open-source-client}

O cliente iOS do Lava é de código aberto. Para dúvidas técnicas, problemas de
compilação ou ideias de recursos, abra um issue em
[**lavasecurity/lavasec-ios**](https://github.com/lavasecurity/lavasec-ios/issues).

## Algo está errado nesta documentação? {#something-wrong-in-these-docs}

Toda página tem, no final, a pergunta **"Esta página foi útil?"** — ao escolher
*"Pode melhorar"*, um issue já preenchido é aberto em
[**lavasecurity/lavasec-doc**](https://github.com/lavasecurity/lavasec-doc/issues).
Correções e pull requests são bem-vindos.
