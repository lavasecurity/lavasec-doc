# 支援與協助 {#support-help}

當你需要協助、發現錯誤或想回報問題時，可以前往這裡。

## 應用程式有功能無法運作？ {#something-not-working-in-the-app}

請使用**應用程式內建的錯誤回報**——這是最快的途徑，並會附上我們所需的診斷資訊，同時在設計上將資料降到最少（絕不會包含你瀏覽的網域）。回報會由團隊分流處理。

## 發現安全性漏洞？ {#found-a-security-vulnerability}

**請勿開立公開議題（issue）。**請寄信至 **security@lavasecurity.app**，並附上問題描述、影響範圍與重現步驟。範圍與揭露方式請參閱 iOS 用戶端的[安全性政策](https://github.com/lavasecurity/lavasec-ios/blob/main/SECURITY.md)。_（地址待確認。）_

## 對開放原始碼用戶端有疑問？ {#question-about-the-open-source-client}

Lava Security 的 iOS 用戶端為開放原始碼。如有技術問題、建置問題或功能構想，歡迎在 [**lavasecurity/lavasec-ios**](https://github.com/lavasecurity/lavasec-ios/issues) 開立議題。

## 這些文件有誤？ {#something-wrong-in-these-docs}

每個頁面底部都有一個 **「這個頁面對你有幫助嗎？」** 的提示——選擇 *「可以更好」* 會在 [**lavasecurity/lavasec-doc**](https://github.com/lavasecurity/lavasec-doc/issues) 開立一個預先填好的議題。歡迎提交修正與 pull request。
